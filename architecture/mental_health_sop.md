# SOP: Mental Health

## Goal
Defines AI stress support.

## Inputs
- user chat messages
- stress level

## Outputs
- stress relief suggestions
- healthy habits

## Logic
1. Analyze text for stress indicators.
2. Determine current reported stress level.
3. Suggest coping mechanisms and healthy habits.

## Edge Cases
- User exhibits signs of self-harm or severe psychological distress. Must trigger crisis helpline fallback.

## Security Rules
- Avoid storing sensitive medical data unnecessarily. Never use real mental health diagnoses.
